# 🎯 学习成果与职业前景地图

学会这套技术栈（Python + LangChain + RAG + Streamlit），你将从一个“会写代码的人”进化为 **“AI 应用工程师”**。

## 1. 你将掌握的 4 大核心能力

| 能力维度 | 你现在的状态 | 学完后的状态 | 核心价值 (为什么企业招你) |
| :--- | :--- | :--- | :--- |
| **AI 交互** | 只会用 ChatGPT 聊天界面 | **能用代码控制 LLM** (API, Temperature, Prompt Template) | 能把 AI 集成到公司现有的软件里。 |
| **私有数据 (RAG)** | 只能问 AI 训练过得通用知识 | **能让 AI 读懂企业私有文档** (PDF, Word, 数据库) | 解决了企业最大的痛点：数据隐私和领域专业性。 |
| **工程落地** | 只有零散的脚本 | **有完整的 Web 界面** (Streamlit) 和 **评测体系** (Eval) | 能做出老板和客户能直接用的产品，而不是半成品。 |
| **智能体 (Agent)** | AI 只能说话 | **AI 能干活** (查天气、搜谷歌、操作 Excel) | 从“咨询顾问”变为“数字员工”。 |

## 2. 你可以独立做出的 3 类项目

学会这些后，你可以轻松构建以下项目（这也是面试时的**杀手级作品**）：

### 🟢 入门级：个人/企业知识库 (当前项目)
*   **功能**: 上传一堆 PDF/TXT（如《员工手册》、《产品说明书》），然后问 AI 相关问题。
*   **场景**:
    *   **企业内部**: "新员工入职助手"（问：饭补多少钱？年假怎么请？）
    *   **个人**: "论文阅读助手"（扔进去 10 篇论文，问：这几篇文章的共同观点是什么？）
*   **用到技术**: RAG, ChromaDB, Embedding.

### 🟡 进阶级：智能客服 / 销售助手
*   **功能**: 放在官网右下角的聊天窗。能回答产品问题，**但绝不胡说八道**（有拒答机制），还能留资（收集用户手机号）。
*   **场景**: 电商售前咨询、SaaS 软件技术支持。
*   **核心竞争力**: **高可信 (High Trust)**。你写的 `refusal_rules.py` 就在这里发挥作用——防止客服乱承诺。
*   **用到技术**: RAG + 拒答策略 + Prompt 优化。

### 🔴 专家级：垂直领域智能体 (Agent)
*   **功能**: 不仅能答，还能**办事**。
*   **场景**:
    *   **"AI 数据分析师"**: 上传 Excel 表格，对它说“帮我画个上季度销售额的柱状图”，它自动写 Python 代码画图给你看。
    *   **"AI 法律顾问"**: 上传合同，它不仅能指出风险条款，还能**自动起草**修改建议并生成 Word 文档。
*   **用到技术**: Agent (Tool Calling), Python REPL, 复杂逻辑编排。

## 3. 你的职业方向

这些技能直接对应目前市场上**薪资最高**的几个岗位：

1.  **AI 应用开发工程师 (AI Application Engineer)**
    *   *日常*: 拿着大模型 API，配合 RAG 技术，给公司开发内部工具或对外产品。
    *   *核心*: 你的 `rag_app.py` 就是这类工作的缩影。

2.  **RAG 算法工程师 (偏应用侧)**
    *   *日常*: 天天研究怎么切分文档 (Chunking) 搜得更准，怎么写 Prompt 让回答更顺滑。
    *   *核心*: 你的 `rag_eval.py` 和 `learning_log.md` 是这类岗位的核心能力。

3.  **AI 产品经理 (技术型)**
    *   *日常*: 懂技术边界，知道 AI 能做什么不能做什么，设计 AI 产品。
    *   *核心*: 你对 Prompt 和 Token 的理解，能让你吊打不懂技术的 PM。
